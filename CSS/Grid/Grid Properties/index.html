Grid Properties : justify-content: value; -> justify-content horizontally(Row vise) Grid ko move karne ke liye hota hai. align-content: value; -> align-content vertically(Column vise) Grid ko move karne ke liye hota hai. justify-items: value; -> justify-items all grid items ko horizontally(Row vise) move karne ke liye hota hai. NOTE: justify-items ka use karne se element ki width uske content jitni ho jaati hai. align-items: value; -> align-items all grid items ko vertically(Column vise) move karne ke liye hota hai. NOTE: align-items ka use karne se element ki height uske content jitni ho jaati hai. justify-self: value; -> justify-self kisi ek grid item ko horizontally move karne ke liye use hota hai. NOTE: justify-self ka use karne se element ki width uske content jitni ho jaati hai. align-self: value; -> align-self kisi ek grid item ko vertically move karne ke liye use hota hai. NOTE: align-self ka use karne se element ki height uske content jitni ho jaati hai. place-content: value; -> place-content hai wo shorthand property hai justify-content OR align-content ki. place-items: value; -> place-items hai wo shorthand property hai justify-items OR align-items ki. place-self: value; -> place-self hai wo shorthand property hai justify-self OR align-self ki.
